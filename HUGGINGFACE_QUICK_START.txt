📋 DEPLOYMENT STEPS
═══════════════════════════════════════════════════════════════════

STEP 1: Create Hugging Face Account (2 min)
─────────────────────────────────────────────
1. Go to: https://huggingface.co
2. Click "Sign Up"
3. Use email or GitHub to sign up
4. Verify your email

STEP 2: Create New Space (2 min)
─────────────────────────────────────────────
1. Click your profile icon → "New Space"
2. Fill in:
   • Space name: ai-text-humanizer
   • License: MIT
   • SDK: Streamlit ⚡
   • Hardware: CPU basic (FREE)
   • Visibility: Public

3. Click "Create Space"

STEP 3: Push to Hugging Face (3 min)
─────────────────────────────────────────────
After creating your space, run these commands:

cd "/home/mehdi/Downloads/other projects/AI-Text-Humanizer-App-main"

# Add Hugging Face remote (replace YOUR_USERNAME!)
git remote add hf https://huggingface.co/spaces/YOUR_USERNAME/ai-text-humanizer

# Push to Hugging Face
git push hf main

⚠️  IMPORTANT: Replace YOUR_USERNAME with your actual Hugging Face username!

STEP 4: Wait for Build (5-8 min)
─────────────────────────────────────────────
Watch the build logs in Hugging Face:
⏳ Installing dependencies...
⏳ Downloading spaCy model...
⏳ Downloading NLTK data...
⏳ Starting Streamlit...
✅ Your app is live!

STEP 5: Access Your App! 🎉
─────────────────────────────────────────────
Your app URL:
https://huggingface.co/spaces/YOUR_USERNAME/ai-text-humanizer
